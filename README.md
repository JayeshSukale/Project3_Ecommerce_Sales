# Project3_Ecommerce_Sales
# Ecommerce Sales Analysis using SQL & Power BI

## Project Overview

This project focuses on analyzing ecommerce sales data using SQL and visualizing key business insights through an interactive Power BI dashboard. The objective was to identify sales trends, customer purchasing behavior, profitable product categories, and regional performance to support data-driven business decisions.

## Objectives

* Analyze sales and profit performance across different states and product categories.
* Identify top customers based on revenue contribution.
* Evaluate monthly sales and profit trends.
* Understand customer payment preferences.
* Calculate key business metrics such as Total Sales, Total Profit, Average Order Value (AOV), and Category Contribution Percentage.

## SQL Analysis Performed

The analysis was conducted using MySQL and covered:

* Data aggregation using GROUP BY and aggregate functions.
* Customer analysis to identify high-value customers.
* State-wise sales and profit analysis.
* Category and sub-category performance analysis.
* Monthly sales trend analysis.
* Payment mode analysis.
* Contribution percentage calculations.
* Customer retention analysis using repeat purchase behavior.
* Business KPI calculations and reporting.

## Power BI Dashboard

An interactive Power BI dashboard was developed to visualize:

* Total Sales
* Total Profit
* Total Quantity Sold
* Monthly Profit Trends
* Sales by State
* Sales by Category
* Profit by Sub-Category
* Payment Mode Distribution
* Customer Performance Metrics

## Tools & Technologies

* SQL (MySQL)
* Power BI
* Data Analysis
* Data Visualization
* Business Intelligence

## Key Insights

* Identified top-performing states contributing the highest revenue.
* Determined the most profitable product categories and sub-categories.
* Analyzed monthly sales patterns to identify seasonal trends.
* Discovered customer segments generating the highest revenue.
* Evaluated payment mode preferences and their impact on sales.

## Project Outcome

The project demonstrates the complete data analysis workflow, from data extraction and SQL-based analysis to interactive dashboard creation, enabling stakeholders to make informed business decisions based on sales and profitability insights.
